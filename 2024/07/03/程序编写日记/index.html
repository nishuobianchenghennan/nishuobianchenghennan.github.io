程序中的类

Atom

Atom 使用的两种方式：

可以将多个原子的信息放在一个列表当中

Ggrid

跟踪用户定义的动能截止参数Ecut所定义的倒数空间网格点集的类

KSSOLV可以处理半径为$r=\sqrt{\text { Ecut }^{2} / 2}$的区域

类属性列表：

ecut ：

一个浮点数，表示用于定义有效倒数空间网格点的能量截止值。

ng：

一个整数，表示 “ecut “截断范围内有效倒数空间网格点的数量

idxnz：

一个数组，用于给出包含所有倒易空间点的三维数组中每个有效倒易空间网格点的索引。该数组也用于 Wavefun 对象。

gkk:

一个数组，给出每个有效倒数空间网格点的幅度平方。

gkx:

一个数组，给出每个有效倒数空间网格点的 x 坐标。

gky:

一个数组，给出每个有效倒数空间网格点的 y 坐标。

gkz :

一个数组，给出每个有效倒易空间网格点的 Z 坐标。

Molecule

class attribution list:

name

指定分子名称的字符串。

supercell

一个 3x3 数组，包含布拉维网格矢量，用于指定包含分子的超胞

alist

包含原子（对象）列表的数组

xyzlist

包含 alist 中每个原子的 x、y、z 坐标的数组。

ecutwfc

一个浮点数，用于指定平面波扩展中使用的动能截止值。

ecutrho

一个浮点数，用于指定计算哈特里(hartree)势时使用的能量截止值。如果未设置此属性，则默认设置为 4*ecutwfc。

n1

x 方向上的网格点数。计算方法是将 Ecut 视为奈奎斯特采样频率。（Nyquist sampling frequency）

$n_{1}=\frac{2 E_{c u t}}{2 \pi / L_{1}}=\frac{E_{c u t} \times L_{1}}{\pi}$

其中，$L_{1}$是沿X方向的单位尺寸

n2

y 方向上的网格点数。计算方法是将 Ecut 视为**奈奎斯特采样频率(Nyquist sampling frequency)**。

$n_{2}=\frac{2 E_{\text {cut }}}{2 \pi / L_{2}}=\frac{E_{\text {cut }} \times L_{2}}{\pi}$

其中，$L_{2}$是沿Y方向的单位尺寸

n3

z方向上的网格点数。计算方法是将Ecut视为**奈奎斯特采样频率(Nyquist sampling frequency)**。

$n_{3}=\frac{2 E_{\text {cut }}}{2 \pi / L_{3}}=\frac{E_{\text {cut }} \times L_{3}}{\pi}$

其中，$L_{3}$是沿Z方向的单位尺寸

gridwfc

gridrho

vext

nspin

指定自旋类型的整数（1 = 自旋非极化，2 = 自旋极化）

temperature

一个浮点数，用于指定有限温度 SCF 计算中使用的温度（单位：加尔文）。

natoms

表示原子个数的整数。

vol

浮点数，表示超胞的体积

nel

一个整数，表示电子（自旋极化）或电子对（自旋非极化）的数量

Crystal

晶体是一个定义周期系单元格的类，除了包含K点属性外，该类与分子类几乎完全相同

name

指定分子名称的字符串。

supercell

一个 3x3 数组，包含布拉维网格矢量，用于指定包含分子的超胞

alist

包含原子（对象）列表的数组

xyzlist

包含 alist 中每个原子的 x、y、z 坐标的数组。

ecutwfc

一个浮点数，用于指定平面波扩展中使用的动能截止值。

ecutrho

一个浮点数，用于指定计算哈特里(hartree)势时使用的能量截止值。如果未设置此属性，则默认设置为 4*ecutwfc。

n1

x 方向上的网格点数。计算方法是将 Ecut 视为奈奎斯特采样频率。（Nyquist sampling frequency）

$n_{1}=\frac{2 E_{c u t}}{2 \pi / L_{1}}=\frac{E_{c u t} \times L_{1}}{\pi}$

其中，$L_{1}$是沿X方向的单位尺寸

n2

y 方向上的网格点数。计算方法是将 Ecut 视为**奈奎斯特采样频率(Nyquist sampling frequency)**。

$n_{2}=\frac{2 E_{\text {cut }}}{2 \pi / L_{2}}=\frac{E_{\text {cut }} \times L_{2}}{\pi}$

其中，$L_{2}$是沿Y方向的单位尺寸

n3

z方向上的网格点数。计算方法是将Ecut视为**奈奎斯特采样频率(Nyquist sampling frequency)**。

$n_{3}=\frac{2 E_{\text {cut }}}{2 \pi / L_{3}}=\frac{E_{\text {cut }} \times L_{3}}{\pi}$

其中，$L_{3}$是沿Z方向的单位尺寸

kpts:

一个 nk 乘 3 的浮点数组，给出布里渊区 k 个点的 x、y、z 坐标。

wks:

一个 nk 乘 1 的浮点数组，给出 kpts 属性中列出的每 k 个点的积分权重

autokpts:

一个 6 乘 1 的浮点数组，前三个元素表示每个倒易点阵方向上的 k 点数，后三个元素表示从原点开始的偏移量。打开该属性时，将生成均匀空间 k 点（蒙霍斯特）。

nspin:

指定自旋类型的整数（1 = 自旋非极化，2 = 自旋极化）

Wavefun

abs, fft3, ifft3, qr, svd, conj

这些函数是重载的，可以直接应用于 Wavefun 对象。它们通常会返回另一个 Wavefun 对象，或一个 Wavefun 对象以及其他矩阵。

wavefun的运算：

​ 我们还可以对两个 Wavefun 对象 X 和 Y 进行加减运算，只需写入 X+Y 或 X-Y。我们可以使用 X.’ 对 X 进行转置，使用 X’ 对 X 进行共轭转置。X 和 Y 的内积可以简单地写成 X’Y。X 和 Y 的哈达玛乘积可以写成 X.Y。X 的每个元素的点幂可以写成 X.^p，其中 p 是整数或浮点数。

​ 可以使用 MATLAB 的子引用符号 X(:,i:j)来提取 X 中的波函数子集，其中 i <= j 是小于或等于 X 中波函数个数的索引。

如果 Wavefun 对象的维度匹配，也可以将它们连接起来，例如 Z = [XY]。

nrows:

整数，用于指定每个波函数的平面波系数个数。

ncols:

一个整数，用于指定对象中包含的波函数个数。

n1:

一个整数，用于指定 x 方向上实际空间网格点的数量。

n2:

一个整数，用于指定 Y 方向上实际空间网格点的数量。

n3:

一个整数，用于指定 Z 方向上实际空间网格点的数量。

idxnz:

一个整数数组，给出了从三维数组重塑为一维矢量时每个波函数的非零平面波系数（存储在 psi 中）的索引。

iscompact:

整数，表示是否只存储每个波函数的非零平面波系数（压缩格式）。如果使用紧凑格式，则应设为 1，否则设为 0。

trans:

一个整数，表示存储的是波函数本身（0）还是其转置（1）

occ:

一个整数数组，给出每个存储波函数的占位编号。

BlochWavefun()

abs, fft3, ifft3, qr, svd, conj

这些函数是重载的，可以直接应用于 Wavefun 对象。它们通常会返回另一个 Wavefun 对象，或一个 Wavefun 对象以及其他矩阵。

我们还可以对两个 Wavefun 对象 X 和 Y 进行加减运算，只需写入 X+Y 或 X-Y。我们可以使用 X.’ 对 X 进行转置，使用 X’ 对 X 进行共轭转置。X 和 Y 的内积可以简单写成 X’Y。X 和 Y 的哈达玛乘积( Hadamard product )可以写成 X.Y。X 的每个元素的点幂可以写成 X.^p，其中 p 是整数或浮点数。

使用 MATLAB 的子引用符号 X(:,i:j)可以提取 X 中的波函数子集，其中 i <= j 是小于或等于 X 中波函数数量的索引。

如果 Wavefun 对象的维度匹配，也可以将它们连接起来，例如 Z = [XY]。

具体的参数信息可参考上一个Wavefun的内容，其是完全一致的

ham

相关参数：
n1

一个整数，用于指定哈密顿在 x 方向上可应用于相应波函数的实空间网格点的数量。

n2

一个整数，用于指定哈密顿在 y 方向上可以应用到的相应波函数的实空间网格点数。

n3

一个整数，用于指定哈密顿在 Z 方向上可以应用到的相应波函数的实空间网格点数。

gkin

包含动能的数组，以傅里叶倒数空间表示波函数

idxnz(与 :attr:Wavefun 对象中使用的相同)

一个整数数组，当波函数从三维数组重塑为一维矢量时，该数组给出了每个波函数的非零平面波系数（存储在 psi 中）的一维索引。

vtot

包含总电势（不包括非局部离子伪电势）的三维阵列。

vion

包含局部离子伪电势的三维阵列。

vnp

包含电荷密度相关电势的三维阵列

vnlmat

包含非局部伪势的 Kleinman-Bylander 投影方向的二维阵列

vnlsign

包含非局部伪势的 Kleinman-Bylander 投影缩放矩阵的二维阵列

rho

三维阵列，包含与哈密顿相关的电子密度。

BlochHam

相关参数：

n1

一个整数，用于指定哈密顿在 x 方向上可应用于相应波函数的实空间网格点的数量。

n2

一个整数，用于指定哈密顿在 y 方向上可以应用到的相应波函数的实空间网格点数。

n3

一个整数，用于指定哈密顿在 Z 方向上可以应用到的相应波函数的实空间网格点数。

nkpts:

一个整数，指定在布里渊区选择的 k 个点的数量。

wks:

一个浮点数组，包含与 K 点相关的权重。这与相应晶体中定义的权重相同。

gkincell:

包含与不同 k 点相关的动能的数组。

idxnz(与 :attr:Wavefun 对象中使用的相同)

一个整数数组，给出每个波函数的非零平面波系数（存储在 psi 中）在大小为 n1 乘 n2 乘 n3 的三维数组中的一维索引。

vtot

包含总电势（不包括非局部离子伪电势）的三维阵列。

vion

包含局部离子伪电势的三维阵列。

vnp

包含电荷密度相关电势的三维阵列

vnlmatcell

包含每个 k 点的非局部伪势的克莱因曼-比兰德（KB）形式的单元阵列。

vnlsigncell

单元数组，包含每 k 点的 KB 形式非局部伪势的缩放因子（矩阵）。

vext

三维数组，包含一个可能由用户定义的外部电势。

rho

三维阵列，包含与哈密顿相关的电子密度。

程序中的算法

平面波离散化

​ KSSOLV 使用平面波展开来表示 Kohn-Sham 特征函数。对于 R 周期 Kohn-Sham 哈密顿，根据布洛赫定理（ Bloch theorem ），特征函数 $\psi_{k}(r)$ 可以表示为:

$\psi_{k}(r)=e^{i k \cdot R} u_{k}(r)$
